# 일정 관리 앱 서버
## 기능 구현
- JDBC를 사용하여 일정을 등록, 조회(전체/선택), 수정, 삭제 할 수 있는 CRUD 기능
- 선택 일정을 수정, 삭제할 때 password를 받아 일치할 경우에만 기능 실행
